### Project Overview
This project contains the analysis I performed on the member data collected via Google Form when **AWS Cloud Club FUTA** was newly established.

The goal was to understand the composition of our early members so we could make data-driven decisions to support effective program planning, targeted outreach, and sustainable community growth.

---

### Objectives
- Analyze member distribution across different faculties and departments
- Understand the level of interest in various Cloud Computing topics
- Identify opportunities and gaps for community engagement
- Provide actionable insights to guide the club’s outreach and programming strategies

---

### Key Insights Delivered
- **Faculty & Department Distribution**: Clear view of which faculties were most represented and which were underrepresented.
- **Cloud Interest Distribution**: Popular cloud topics and technologies among members.
- **Demographic Patterns**: Helped identify potential focus areas for growth.
- **Strategic Recommendations**: Insights used to shape outreach efforts, event planning, and content strategy for the club.

---

### What’s Inside

- `AWS Cloud Club,FUTA.csv` – Raw response data 
- `AWS Cloud Club FUTA Cleaned/` – Cleaned excel file
- `AWS Cloud Club FUTA Dashboard` – Charts and graphs showing member distribution and interests
- `AWS Cloud Club,FUTA.docx` – Detailed insights and recommendations for club leadership

---

### Technologies Used

---

### Impact
The findings from this analysis directly informed:
- Targeted outreach campaigns to underrepresented faculties
- Selection of technical session topics
- Strategy for community growth in the early days of the club
